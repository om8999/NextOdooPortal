## NextOdooPortal

## Overview

OdooPortal is a sleek, modern web portal built with **Next.js** for the frontend and **FastAPI** as a middleware layer, designed to act as a **secure login and admin interface for Odoo**. It provides a polished user experience with a **glassmorphic, neon-glow UI**, while offering a robust middleware layer to handle API requests, authentication, and CORS for Odoo.

---

## Features

### User Authentication
- Signup, login, and secure password handling  
- Password-protected actions (like user deletion)

### Admin Dashboard
- User management with role-based access  
- CRUD operations for users  
- Secure admin-only routes

### Modern UI/UX
- Glassmorphic cards, gradient backgrounds, and hover effects  
- Responsive and mobile-friendly design  
- Neon-glow accents for a modern SaaS look

### Middleware Layer
- FastAPI handles API requests, validation, and CORS  
- Acts as a protective layer for Odoo backend endpoints

### Scalable & Modular
- Easily extendable to include more Odoo modules  
- Modular frontend and backend structure
  
## Tech Stack
- **Frontend:** Next.js, React, Tailwind CSS  
- **Middleware/API:** FastAPI (Python)  
- **Backend:** Odoo ERP
